# Plantillas Canva - FOGATTA

Estas plantillas estan en SVG para usarlas como guia dentro de Canva.

## Archivos

1. `fondo-desktop-1920x1080.svg`
2. `fondo-mobile-1080x1920.svg`
3. `hero-1200x1200.svg` (incluye zona segura 900x900 centrada)

## Como usarlas en Canva

1. Crea un diseno con tamano personalizado segun el archivo.
2. Sube el SVG correspondiente.
3. Ponlo como capa base y bloquealo.
4. Disena encima respetando bordes y zona segura.
5. Exporta en WebP (si Canva plan lo permite) o PNG/JPG.

## Medidas oficiales

1. Fondo global desktop: 1920 x 1080 px
2. Fondo global mobile: 1080 x 1920 px
3. Hero: 1200 x 1200 px
4. Zona segura hero: 900 x 900 px centrada
